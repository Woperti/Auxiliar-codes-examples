## Crear modulos reutilizables

```Java
import javax.swing.*;
import java.awt.*;

public class Button extends JButton {
  public Button(String text) {
    super(text);
    setFont(new Font("Arial", Font.BOLD, 14);
    setBackground(Color.LIGHT_Gray);
    setForeground(Color.BLACK);
    setFocusPainted(false);
  }
}
```
